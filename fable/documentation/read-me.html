3CSS Structure

This template also includes css files. Let's take a look closely at our stylesheet files:

  • bootstrap.css - standard bootstrap css styles.
  • style.css - common css styles for this template.
  • animate.css- a simple CSS library that saves you from writing a few more lines of CSS to animate elements on your website.
  • style-magnific-popup - contain css styles for the magnific-popup lightbox.
